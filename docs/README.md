Эта папка — **единственный источник истины** по дизайну, плану и правилам проекта.

документ, потом реализация. История переноса из донора — отдельно, в

## Как этим пользоваться (мне и агентам)

1. **Начало любой сессии:** открыть [04_ROADMAP.md](04_ROADMAP.md), найти первый
   незакрытый этап, взять из него задачи. Контекст по механикам — в GDD,
   по редакторским шагам — в EDITOR_GUIDE.
2. **Постановка задачи агенту:** «Работаем по docs/04_ROADMAP.md, этап N, задачи
   такие-то. Правила — docs/06_CONVENTIONS.md». Этого достаточно: документы
   самодостаточны.
3. **Любое отклонение от GDD** (упростили механику, поменяли число) — сразу
   фиксируется в GDD, чтобы документ не разошёлся с игрой.
4. **После выполнения** — проставить чекбоксы в ROADMAP и, если менялся C++,
   обновить беклог в 03_CODE_OVERVIEW.

- **Кодовая база закрывает весь GDD** (этап 1 выполнен): управление боем
  (контроллер+камера+зона хода), атака/овервотч/оборона/лечение/рывок/
  провокация, смерть/ранение/эвакуация, бомба+таймер, AI с тревогой
  Patrol/Investigate/Combat, меню/сейвы/HUD-базы. Карта кода — 03_CODE_OVERVIEW.
- **В редакторе поднято управление боем** (на тестовой карте Lvl_TopDown):
  BP камеры/контроллера/визуализатора/юнитов + GM_Tactics + Enhanced Input.
  Выбор/движение/камера работают; зона хода и превью пути отрисовываются точно
  (волновой Dijkstra + marching squares + лента по рельефу).
- **Подсветка юнитов (C++ готов, 2026-07-15):** ховер-обводка через Custom Depth
  Stencil (post-process) + кольцо-декаль под выбранным юнитом. В редакторе
  осталось создать M_OutlinePP и M_SelectionRing и назначить в BP —
  см. 05_EDITOR_GUIDE §4.6.
- **Камера и навмеш (C++, 2026-07-16):** XCOM-камера — полёт/следование за
  бойцом и действующим врагом (LOS-гейт), edge scroll. Юниты — динамические
  препятствия навмеша (пути/зоны их огибают; нужен RecastNavMesh Runtime
  Generation = Dynamic).

  (ховер-цель, счётчик врагов, контекст F, позывные, лимиты способностей) и
  виджеты над головой (пипсы AP, иконка укрытия). Следующий шаг — **сборка
